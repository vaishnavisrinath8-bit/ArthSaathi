# ArthSaathi Demo Script

## 1. Onboarding

- Open the app.
- Select a language.
- Show both entry options:
  - Create new account
  - Login to existing account

## 2. Signup

- Create a new account with name, mobile number, password.
- Choose a profession:
  - Farmer
  - Grocery Shop
  - Tailor
  - Daily Wage

## 3. Profession Questions

- Show the question page after signup.
- Switch between Normal input and Voice input.
- Use simulated voice answers to fill sample data.
- Forward to dashboard.

## 4. Home Dashboard

- Show financial health card.
- Show income, expense, savings.
- Show loan risk card.
- Show universal quick actions: Add Expense and Loan Risk.
- Show profession-specific quick actions.

## 5. Ledger

- Open Ledger.
- Show that transaction categories change by profession.
- Add a local transaction.
- Show lending and borrowing tabs.

## 6. Profession Dashboard

Open profession quick action:

- Shop: Add Udhar and Add Stock are separate.
- Tailor: Add Order and Add Delivery are separate.
- Daily Wage: Add Shift and Add Payment Due are separate.
- Farmer: Mandi rates and RTC readiness.

## 7. Loan

- Open Loan Risk Check.
- Select a profession-specific purpose chip.
- Run analysis.
- Show risk gauge, EMI, and total repayment.

## 8. Profile

- Show improved profile header.
- Show income/expense and work profile.
- Show language and preferences.
- Sign out returns to onboarding.
